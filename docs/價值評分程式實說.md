## 1. 主函式架構

**檔案位置**: `core/scorer.py` → `def score_value(row, data_years_available=10) -> dict`（第 1197 行）

```
score_value(row, data_years_available)
    ├── score_valuation_safety(row)       # 權重 15%
    ├── score_profit_quality(row)         # 權重 20%
    ├── score_growth_ability(row)         # 權重 30%
    ├── score_financial_safety_value(row) # 權重 15% (金融業跳過)
    ├── score_cash_flow_quality(row)      # 權重 10% (金融業跳過)
    └── score_shareholder_return(row)     # 權重 10%
              ↓
       加權總分 (base_score)
              ↓
    apply_all_modifiers(base_score, row, "value", dq_modifier, industry_bias)
              ↓
       最終價值總分 (0-100)
```

**權重設定檔**: `core/scoring_config.py` 第 190-197 行

```python
VALUE_WEIGHTS = {
    "valuation_safety": 0.15,
    "profit_quality": 0.20,
    "growth_ability": 0.30,
    "financial_safety": 0.15,
    "cash_flow_quality": 0.10,
    "shareholder_return": 0.10,
}
```

**回傳格式**:

```python
{
    "total": int,
    "breakdown": {
        "valuation_safety": int,
        "profit_quality": int,
        "growth_ability": int,
        "financial_safety": int,
        "cash_flow_quality": int,
        "shareholder_return": int,
    },
    "details": {
        "valuation_safety": {...},
        "profit_quality": {...},
        ...
    }
}
```

## 2. 評分流程總覽

```
【輸入】row (dict), data_years_available (int)
         │
         ▼
   是否為金融業？ ──是──→ 跳過 financial_safety + cash_flow_quality
         │
         否
         ▼
   6 個子項各自獨立評分 (0-100)
         │
         ▼
   金融業權重再分配（其餘 4 子項等比例放大）
         │
         ▼
   加權計算 → base_score = Σ(子項分數 × 權重)
         │
         ▼
   apply_all_modifiers()
    ├── Data Quality Modifier
    ├── Industry Debt Bias (負債比 > 同業中位數×1.2 打 85 折)
    └── Risk Modifier
         │
         ▼
   np.clip(0, 100) → 最終價值總分
```

## 3. 子項評分函式

### 3.1 估值安全（valuation_safety）

**檔案**: `core/scorer.py` 第 897 行 `def score_valuation_safety(row)`

**評分方式**: PE_TTM 絕對值 + PB_Percentile 雙條件（Phase 2 重構）

#### 有 PE_TTM 和 PB_Percentile：

| PE_TTM | PB_Percentile | 評分 |
|:------:|:-------------:|:----:|
| ≤ 12 | ≤ 20% | **100** |
| ≤ 15 | ≤ 40% | **85** |
| ≤ 20 | 或 ≤ 60% | **70** |
| ≤ 30 | 且 ≤ 80% | **50** |
| > 30 或 > 80% | | **0** |

#### 僅有 PE_TTM 時：

| PE_TTM | 評分 |
|:------:|:----:|
| ≤ 12 | 85 |
| ≤ 20 | 70 |
| ≤ 30 | 50 |
| > 30 | 0 |

#### 僅有 PB_Percentile 時：

| PB_Percentile | 評分 |
|:-------------:|:----:|
| ≤ 20% | 85 |
| ≤ 40% | 70 |
| ≤ 60% | 50 |
| > 60% | 0 |

#### 例外處理

- 兩者皆無資料 → score=70（中性）
- `pe_ratio` 為 NaN → 跳過 PE 條件
- `PB_Percentile` 為 NaN → 跳過 PB 條件

---

### 3.2 獲利品質（profit_quality）

**檔案**: `core/scorer.py` 第 958 行 `def score_profit_quality(row)`

**評分方式**: ROE_TTM + Gross_Margin 雙條件（Phase 2 重構）

| ROE_TTM | Gross_Margin | 評分 |
|:-------:|:------------:|:----:|
| ≥ 15% | ≥ 50% | **100** |
| ≥ 10% | ≥ 30% | **85** |
| ≥ 6% | — | **70** |
| ≥ 3% | — | **50** |
| < 3% 或 NaN | — | **0** |

#### 門檻設定（scoring_config.py 第 212-227 行）

```python
# roe_* 為 ROE 門檻、gm_* 為 Gross_Margin 門檻
"roe_excellent": 20, "gm_excellent": 50,
"roe_good": 15,     "gm_good": 30,
"roe_normal": 8,    "gm_normal": 15,
"roe_weak": 3,      "gm_weak": 5,
"roe_poor": 0,      "gm_poor": 0,
```

> 程式碼實際使用硬編碼門檻 `roe >= 15`、`roe >= 10`、`roe >= 6`、`roe >= 3`，並未從 config 讀取。

#### 例外處理

- `ROE_TTM` 為 NaN → 回傳 score=0，note="ROE_TTM 資料不足"
- `Gross_Margin` 為 NaN → 不影響 ROE 條件，但無法觸發 100/85 分

### 3.3 成長能力（growth_ability）

**檔案**: `core/scorer.py` 第 1003 行 `def score_growth_ability(row)`

**評分方式**:

```
composite = TTM_EPS × 60% + Revenue_YoY × 40%
```

#### 子指標 1：TTM EPS（近四季每股盈餘）

| 值 | 評分 |
|:---:|:----:|
| ≥ 15 | 100 |
| ≥ 8 | 85 |
| ≥ 3 | 70 |
| ≥ 0 | 50 |
| < -5 | 0 |

#### 子指標 2：營收年增率（Revenue_YoY）

| 值 | 評分 |
|:---:|:----:|
| ≥ 20% | 100 |
| ≥ 10% | 85 |
| ≥ 5% | 70 |
| ≥ 0% | 50 |
| < -20% | 0 |

#### 門檻設定（scoring_config.py 第 229-239 行）

```python
"ttm_eps_excellent": 15, "rev_yoy_excellent": 20,
"ttm_eps_good": 8,      "rev_yoy_good": 10,
"ttm_eps_normal": 3,    "rev_yoy_normal": 5,
"ttm_eps_weak": 0,      "rev_yoy_weak": 0,
"ttm_eps_poor": -5,     "rev_yoy_poor": -20,
```

#### 例外處理

- `TTM_EPS` 為 NaN 或 `TTM_EPS_Valid = False` → `eps_score = 0`
- `Revenue_YoY` 為 NaN → 代入 0

### 3.4 財務安全（financial_safety）

**檔案**: `core/scorer.py` 第 1043 行 `def score_financial_safety_value(row)`

**評分方式**: **金融業跳過**，給基準滿分 100

```
composite = Debt_Ratio(反) × 60% + Current_Ratio × 40%
```

#### 子指標 1：負債比（反向評分）

| 值 | 評分 |
|:---:|:----:|
| ≤ 30% | 100 |
| ≤ 45% | 85 |
| ≤ 60% | 70 |
| ≤ 75% | 50 |
| > 90% | 0 |

#### 子指標 2：流動比率（正向評分）

| 值 | 評分 |
|:---:|:----:|
| ≥ 2.5 | 100 |
| ≥ 2.0 | 85 |
| ≥ 1.5 | 70 |
| ≥ 1.0 | 50 |
| < 0.5 | 0 |

#### 門檻設定（scoring_config.py 第 248-258 行）

```python
"debt_ratio_excellent": 30,  "current_ratio_excellent": 2.5,
"debt_ratio_good": 45,      "current_ratio_good": 2.0,
"debt_ratio_normal": 60,    "current_ratio_normal": 1.5,
"debt_ratio_weak": 75,      "current_ratio_weak": 1.0,
"debt_ratio_poor": 90,      "current_ratio_poor": 0.5,
```

#### 例外處理

- 金融業 (`is_finance(row)`) → 直接回傳 score=100
- `Debt_Ratio` 預設值為 100（`row.get("Debt_Ratio", 100)`）

```
composite = TTM_FCF × 60% + TTM_OCF × 40%
```

#### 子指標 1：TTM_FCF（近四季自由現金流）

|:--------:|:----:|

| ≥ 50 億 | 85 |
| ≥ 0 | 70 |
| ≥ -50 億 | 50 |
| < -200 億 | 0 |

```python
"ttm_fcf_excellent": 10000000000, "ocf_excellent": 20000000000,
```

#### 例外處理

- 金融業 (`is_finance(row)`) → 直接回傳 score=100

### 3.6 股東報酬（shareholder_return）

**檔案**: `core/scorer.py` 第 1140 行 `def score_shareholder_return(row)`

**評分方式**:

```
composite = dividend_yield × 60% + 有無配息 × 40%
```

#### 子指標 1：殖利率（dividend_yield）

| 值 | 評分 |
|:---:|:----:|
| ≥ 5% | 100 |
| ≥ 3% | 85 |
| ≥ 1.5% | 70 |
| ≥ 0.5% | 50 |
| < 0.5% | 0 |

#### 子指標 2：有無配息（cash_dividend_total）

| 值 | 評分 |
|:---:|:----:|
| > 0（有配息） | 100 |
| = 0（無配息） | 0 |
| NaN（無資料） | 70 分（中性） |

#### 門檻設定（scoring_config.py 第 272-277 行）

```python
"div_yield_excellent": 5,
"div_yield_good": 3,
"div_yield_normal": 1.5,
"div_yield_weak": 0.5,
"div_yield_poor": 0,
```

## 4. 金融業特殊處理

### 4.1 跳過子項

| 跳過子項 | 原始權重 |
|:---------|:--------:|
| financial_safety | 15% |
| cash_flow_quality | 10% |
| **合計** | **25%** |

### 4.2 權重再分配

透過 `_redistribute_weights()`（第 1176 行）將跳過的權重等比例放大到其餘 4 子項：

```python
ratio = 1.0 / (1.0 - 0.25) = 1.3333
valuation_safety:     15% × 1.3333 = 20%
profit_quality:       20% × 1.3333 = 26.7%
growth_ability:       30% × 1.3333 = 40%
shareholder_return:   10% × 1.3333 = 13.3%
```

### 4.3 Industry Debt Bias

適用於價值/定存風格。若負債比 > 同業中位數 × 1.2，額外打 85 折。

```python
INDUSTRY_DEBT_BIAS = {
    "exclude_sectors": ["金融業", "營建業"],
    "debt_ratio_multiplier": 1.2,
    "value_penalty": 0.85,
    "dividend_penalty": 0.85,
}
```

## 5. Modifier 疊加

與短線相同，統一透過 `apply_all_modifiers()` 處理。

### 5.1 Data Quality Modifier

| 資料年數 | Modifier |
|:--------:|:--------:|
| ≥ 8 年 | × 1.00 |
| ≥ 5 年 | × 0.95 |
| ≥ 3 年 | × 0.85 |
| < 3 年 | × 0.70 |

### 5.2 Risk Penalty

| 條件 | Penalty |
|:-----|:-------:|
| 負債比 > 70%（非金融業） | -10 |
| TTM_EPS < 0 | -15 |
| TTM_OCF < 0 且 Payout_Ratio > 100%（非金融業） | -15 |

### 5.3 Risk Bonus

| 條件 | Bonus |
|:----|:-----:|
| RSI_6 < 30 | +5 |
| 負債比 < 20%（非金融業） | +5 |
