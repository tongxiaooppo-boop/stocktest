## 1. 主函式架構

**檔案位置**: `core/scorer.py` → `def score_dividend(row, data_years_available=10) -> dict`（約第 1500 行附近）

```
score_dividend(row, data_years_available)
    ├── score_dividend_record(row)           # 權重 25%
    ├── score_dividend_quality(row)          # 權重 20%
    ├── score_cash_flow_dividend(row)        # 權重 20% (金融業跳過)
    ├── score_financial_safety_dividend(row) # 權重 15% (金融業跳過)
    ├── score_profit_stability(row)          # 權重 10%
    └── score_long_term_growth(row)          # 權重 10%
              ↓
       加權總分 (base_score)
              ↓
    apply_all_modifiers(base_score, row, "dividend", dq_modifier, industry_bias)
              ↓
       最終定存總分 (0-100)
```

**權重設定檔**: `core/scoring_config.py` 第 281-288 行

```python
DIVIDEND_WEIGHTS = {
    "dividend_record": 0.25,
    "dividend_quality": 0.20,
    "cash_flow": 0.20,
    "financial_safety": 0.15,
    "profit_stability": 0.10,
    "long_term_growth": 0.10,
}
```

## 2. 評分流程總覽

```
【輸入】row (dict), data_years_available (int)
         │
         ▼
   是否為金融業？ ──是──→ 跳過 cash_flow + financial_safety
         │
         否
         ▼
   6 個子項各自獨立評分 (0-100)
         │
         ▼
   金融業權重再分配（其餘 4 子項等比例放大）
         │
         ▼
   加權計算 → base_score = Σ(子項分數 × 權重)
         │
         ▼
   apply_all_modifiers()
    ├── Data Quality Modifier
    ├── Industry Debt Bias (負債比 > 同業中位數×1.2 打 85 折)
    └── Risk Modifier
         │
         ▼
   np.clip(0, 100) → 最終定存總分
```

## 3. 子項評分函式

### 3.1 配息紀錄（dividend_record）

**檔案**: `core/scorer.py` 第 1277 行 `def score_dividend_record(row)`

**評分方式**: 連續配息年數 + 殖利率雙條件（Phase 2 重構）

| 連續配息年數 | 殖利率 | 評分 |
|:----------:|:------:|:----:|
| ≥ 10 年 | ≥ 6.0% | **100** |
| ≥ 7 年 | ≥ 4.5% | **85** |
| ≥ 5 年 | ≥ 3.0% | **70** |
| ≥ 3 年 | — | **50** |
| < 3 年 或 無配息 | — | **0** |

#### 變數說明

| 變數 | 資料來源 |
|:-----|:---------|
| `Dividend_Continuity_Years` | processor 計算（連續配息不中斷年數） |
| `dividend_yield` | processor 計算（殖利率 = 現金股利總額 / 股價） |

#### 例外處理

- `Dividend_Continuity_Years` 為 NaN 或 0 → 回傳 score=0，note="無配息紀錄"
- `dividend_yield` 為 NaN → 不影響年數條件，但無法觸發 100/85/70 分

---

### 3.2 配息品質（dividend_quality）

**檔案**: `core/scorer.py` 第 1327 行 `def score_dividend_quality(row)`

**評分方式**: 配息率區間 + EPS Cover Ratio 雙條件

**EPS Cover 計算公式**:
```
EPS Cover = TTM_EPS / cash_dividend_total
```

| 條件 | 評分 | 說明 |
|:-----|:----:|:-----|
| Payout_Ratio ∈ [60%, 80%] 且 EPS Cover ≥ 2.0 | **100** | 配息率適中、盈餘充足 |
| Payout_Ratio ∈ [45%, 60%) 或 EPS Cover ≥ 1.5 | **85** | 配息略低或盈餘良好 |
| Payout_Ratio ∈ [30%, 45%) 或 (80%, 90%] | **70** | 略保守或略高 |
| Payout_Ratio < 30% 或 EPS Cover < 1.0 | **50** | 配息過低或盈餘不足 |
| Payout_Ratio > 90% 或 資料不足 | **0** | 配息過高或無法判斷 |

#### 門檻設定（scoring_config.py 第 298-310 行）

```python
"payout_ratio_excellent": 60,  "payout_ratio_good_low": 40, ...
"eps_cover_excellent": 2.0,   "eps_cover_good": 1.5,
"eps_cover_normal": 1.0,      "eps_cover_weak": 0.5,
```

#### 例外處理

- `Payout_Ratio` 為 NaN → 回傳 score=0，note="配息率資料不足"
- `TTM_EPS` 或 `cash_dividend_total` 為 NaN，或 `cash_dividend_total ≤ 0`，或 `TTM_EPS ≤ 0` → `eps_cover` 無法計算，不影響但無法觸發 +100/85 分條件

---

### 3.3 現金流（cash_flow）

**檔案**: `core/scorer.py` 第 1378 行 `def score_cash_flow_dividend(row)`

**評分方式**: OCF/NI 現金轉換率 + FCF 正數條件（Phase 2 重構）

**Cash_Conv_Ratio 計算公式**:
```
Cash_Conv_Ratio = TTM_OCF / TTM_NetIncome
```

| 條件 | 評分 |
|:-----|:----:|
| Cash_Conv_Ratio ≥ 100% 且 FCF > 0 | **100** |
| Cash_Conv_Ratio ≥ 80% 且 FCF > 0 | **85** |
| Cash_Conv_Ratio ≥ 50% | **70** |
| Cash_Conv_Ratio ≥ 0% | **50** |
| Cash_Conv_Ratio < 0% 或 NI ≤ 0 或 資料不足 | **0** |

#### 門檻設定（scoring_config.py 第 312-317 行）

```python
"fcf_cover_excellent": 2.0, "fcf_cover_good": 1.5,
"fcf_cover_normal": 1.0,   "fcf_cover_weak": 0.5,
```

#### 例外處理

- `TTM_NetIncome` 為 NaN 或 ≤ 0 → 回傳 score=0，note="淨利 <= 0，無法計算現金轉換率"
- `TTM_OCF` 為 NaN → 回傳 score=0，note="TTM_OCF 資料不足"
- `TTM_FCF` 為 NaN → `fcf_positive = False`

---

### 3.4 財務安全（financial_safety）

**檔案**: `core/scorer.py` 第 1440 行 `def score_financial_safety_dividend(row)`

**評分方式**: **金融業跳過**，給基準滿分 100

```
composite = Debt_Ratio(反) × 60% + Interest_Coverage × 40%
```

#### 子指標 1：負債比（反向評分）

| 值 | 評分 |
|:---:|:----:|
| ≤ 30% | 100 |
| ≤ 45% | 85 |
| ≤ 60% | 70 |
| ≤ 75% | 50 |
| > 90% | 0 |

#### 子指標 2：利息保障倍數（正向評分）

| 值 | 評分 |
|:---:|:----:|
| ≥ 10 | 100 |
| ≥ 5 | 85 |
| ≥ 3 | 70 |
| ≥ 1 | 50 |
| < 1 | 0 |

#### 門檻設定（scoring_config.py 第 319-329 行）

```python
"debt_ratio_excellent": 30,    "interest_cover_excellent": 10,
"debt_ratio_good": 45,        "interest_cover_good": 5,
"debt_ratio_normal": 60,      "interest_cover_normal": 3,
"debt_ratio_weak": 75,        "interest_cover_weak": 1,
"debt_ratio_poor": 90,        "interest_cover_poor": 0,
```

#### 例外處理

- 金融業 (`is_finance(row)`) → 直接回傳 score=100
- `Debt_Ratio` 預設值為 100（`row.get("Debt_Ratio", 100)`）
- `Interest_Coverage` 預設值為 0

### 3.5 獲利穩定（profit_stability）

**檔案**: `core/scorer.py` 第 1490 行 `def score_profit_stability(row)`

**評分方式**: **反向評分** — ROE/EPS 波動越低越好

```
composite = ROE_Std(反) × 50% + EPS_Std(反) × 50%
```

#### 子指標 1：ROE 穩定度（ROE_Stability，標準差，反向評分）

| 值 | 評分 |
|:---:|:----:|
| ≤ 3% | 100 |
| ≤ 5% | 85 |
| ≤ 8% | 70 |
| ≤ 12% | 50 |
| > 20% | 0 |

#### 子指標 2：EPS 穩定度（反向評分）

| 值 | 評分 |
|:---:|:----:|
| ≤ 2 | 100 |
| ≤ 4 | 85 |
| ≤ 6 | 70 |
| ≤ 10 | 50 |
| > 20 | 0 |

#### 門檻設定（scoring_config.py 第 331-341 行）

```python
"roe_std_excellent": 3,   "eps_std_excellent": 2,
"roe_std_good": 5,        "eps_std_good": 4,
"roe_std_normal": 8,      "eps_std_normal": 6,
"roe_std_weak": 12,       "eps_std_weak": 10,
"roe_std_poor": 20,       "eps_std_poor": 20,
```

#### 例外處理

- `ROE_Stability` 為 NaN → `roe_std = 999`，分數偏低
- `EPS_Stability` 為 NaN → `eps_std = 999`，分數偏低

---

### 3.6 長期成長（long_term_growth）

**檔案**: `core/scorer.py` 第 1500 行後 `def score_long_term_growth(row)`

**評分方式**:

```
composite = Revenue_CAGR × 50% + EPS_YoY × 50%
```

#### 子指標 1：營收 CAGR（年複合成長率）

| 值 | 評分 |
|:---:|:----:|
| ≥ 15% | 100 |
| ≥ 10% | 85 |
| ≥ 5% | 70 |
| ≥ 0% | 50 |
| < -20% | 0 |

#### 子指標 2：EPS YoY（EPS 年成長率）

| 值 | 評分 |
|:---:|:----:|
| ≥ 30% | 100 |
| ≥ 15% | 85 |
| ≥ 5% | 70 |
| ≥ 0% | 50 |
| < -20% | 0 |

#### 門檻設定（scoring_config.py 第 343-353 行）

```python
"rev_cagr_excellent": 15,   "eps_yoy_excellent": 30,
"rev_cagr_good": 10,        "eps_yoy_good": 15,
"rev_cagr_normal": 5,       "eps_yoy_normal": 5,
"rev_cagr_weak": 0,         "eps_yoy_weak": 0,
"rev_cagr_poor": -20,       "eps_yoy_poor": -20,
```

#### 例外處理

- `Revenue_CAGR` 預設值為 0
- `EPS_YoY` 不可用（denominator_invalid / insufficient_history）→ `eps_yoy_score = 0`

## 4. 金融業特殊處理

### 4.1 跳過子項

| 跳過子項 | 原始權重 |
|:---------|:--------:|
| cash_flow | 20% |
| financial_safety | 15% |
| **合計** | **35%** |

### 4.2 權重再分配

透過 `_redistribute_weights()` 將跳過的權重等比例放大到其餘 4 子項：

```python
ratio = 1.0 / (1.0 - 0.35) = 1.5385
dividend_record:    25% × 1.5385 = 38.5%
dividend_quality:  20% × 1.5385 = 30.8%
profit_stability:  10% × 1.5385 = 15.4%
long_term_growth:  10% × 1.5385 = 15.4%
```

### 4.3 Industry Debt Bias

與價值風格相同。若負債比 > 同業中位數 × 1.2，額外打 85 折。

```python
INDUSTRY_DEBT_BIAS = {
    "exclude_sectors": ["金融業", "營建業"],
    "debt_ratio_multiplier": 1.2,
    "dividend_penalty": 0.85,
}
```

## 5. Modifier 疊加

與其他風格相同，統一透過 `apply_all_modifiers()` 處理。

### 5.1 Data Quality Modifier

| 資料年數 | Modifier |
|:--------:|:--------:|
| ≥ 8 年 | × 1.00 |
| ≥ 5 年 | × 0.95 |
| ≥ 3 年 | × 0.85 |
| < 3 年 | × 0.70 |

### 5.2 Risk Penalty

| 條件 | Penalty |
|:-----|:-------:|
| 負債比 > 70%（非金融業） | -10 |
| TTM_EPS < 0 | -15 |
| TTM_OCF < 0 且 Payout_Ratio > 100%（非金融業） | -15 |

### 5.3 Risk Bonus

| 條件 | Bonus |
|:----|:-----:|
| RSI_6 < 30 | +5 |
| 負債比 < 20%（非金融業） | +5 |
